Simple Js App(Pokedox app)

Simple Js App is a web application that uses HTML, CSS, Javascript, and Bootstrap to display detailed information about Pokemon. Meet App

Meet app is a React-based serverless PWA developed using test-driven development. It utilizes the Google Calendar API to retrieve upcoming events, allowing users to view events from all cities or select a specific city from suggestions. The app also includes features such as event filtering based on number of events and data visualization for displaying event details.

Tech-stack:Javascript,React,React Testing libraries:Jest,Cucumber,Puppeteer, OAuth2,AWS Lambda functions, Recharts
