Simple Js App(Pokedox app)

Simple Js App is a Pokemon small web application made using HTML, CSS, Javascript and Bootstrap that loads data from external API and enables viewing of pokemon data in detail. When you click on any pokemon name a modal will open containing height, weight, type and larger image of pokemon.It also include feature to search pokemon by its name from fetched pokemon list.

Tech-stack: HTML, CSS, Bootstrap, JavaScript, Code Linters used are : HTMLHINT, stylelint, ESLint, Prettier

Meet App

Meet app is a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events and enables viewing upcoming events from all cities or user can select one particular city from the list of suggestion. It also includes features like filter the events based on number of events and data visualization technique to display event details.

Tech-stack:Javascript,React,React Testing libraries:Jest,Cucumber,Puppeteer, OAuth2,AWS Lambda functions, Recharts
